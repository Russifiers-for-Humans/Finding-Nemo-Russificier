ГЛАВНОЕ МЕНЮ

New Game (Новая игра):

  • начать новое приключение
  • выбрать любой из трех предложенных слотов для сохраняемых игр

Load (Загрузить):

  • загрузить ранее сохраненную игру

Delete (Удалить):

  • удалить ранее сохраненную игру

Options (Настройки):

  • настроить параметры игры (звуковое сопровождение, управляющие клавиши и т. д.)

Exit (Выход):

  • выйти из игры

МЕНЮ НАСТРОЕК

Здесь можно установить параметры звука и видео, поменять управляющие клавиши и язык сопровождения.

ОСНОВНОЙ ЭКРАН ИГРЫ

Во время игры на экране отображаются следующие элементы:

  • Жизни — число оставшихся жизней. Выражение лица Дональда отражает его текущее настроение.
  • Звезды — общее число собранных звезд.
  • Время — отсчет времени (появляется только в игре на время).
  • Игрушки — число собранных игрушек (появляется только в игре «Игрушки племянников»).
  • Книга заклинаний — время, оставшееся для того, чтобы взять игрушки.

ЭКРАН НАХОДОК

Этот экран появится после нажатия клавиши I на клавиатуре в ходе игры на уровне. Он отображает количество собранных на уровне игрушек. Здесь же можно увидеть, найдена или нет на текущем уровне часть босс-порта.

ПАУЗА

Этот экран появится после нажатия клавиши ESC.

Resume game (Продолжить игру):

  • возобновить прерванную игру

Load (Загрузить):

  • загрузить ранее сохраненную игру

Option (Настройки):

  • перейти в меню настроек

Gyro's Lab (Лаборатория Винта):

  • перейти в лабораторию Винта

ЛАБОРАТОРИЯ ВИНТА

ГАММА-ТРУБНЫЙ ТЕЛЕПОРТ

Это невероятное изобретение! Оно открывает игроку доступ к уровням и мирам.

КОСТЮМОСМЕННЫЙ АППАРАТ

Эта машина позволяет Дональду сменить костюм (новую одежду можно найти на уровнях игры).

Если нажать клавишу ESC, когда Дональд находится в лаборатории, откроется следующее меню:

Save (Сохранить):

  • сохранить игру

Load (Загрузить):

  • загрузить ранее сохраненную игру

Delete (Удалить):

  • удалить ранее сохраненную игру

Options (Настройки):

  • перейти в меню настроек

Abandon (Прекратить):

  • вернуться в главное меню
