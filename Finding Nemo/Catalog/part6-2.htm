ГЛАВНОЕ МЕНЮ

Чтобы выбрать один из пунктов меню, перейдите на соответствующую строку с помощью клавиш СТРЕЛКА ВВЕРХ и СТРЕЛКА ВНИЗ и нажмите ENTER.

ADVENTURE (БОЛЬШОЕ ПРИКЛЮЧЕНИЕ)

Чтобы починить сломавшуюся машину для фейерверков, вы должны найти все ее детали. Но вы получите их, только если станете победителем! Войдя в раздел ADVENTURE, вы увидите меню начала игры.

VS (МНОГОПОЛЬЗОВАТЕЛЬСКИЙ РЕЖИМ)

Режим VS включается только в том в случае, если к компьютеру подключен игровой пульт. При этом один игрок играет с клавиатуры, а второй – с игрового пульта. Вы можете соревноваться на любой из трасс на скорость или собирать монетки на время. Для настройки клавиш управления в многопользовательском режиме выберите в главном меню пункт Options (Настройки), а затем Controls (Настройка управления).

Чтобы начать игру в многопользовательском режиме:

  1. В главном меню выберите VS и нажмите клавишу ENTER.
  2. Сначала гонщика выбирает игрок № 1, потом игрок № 2 (вверху экрана появится надпись player 2). Игроки не могут выбрать одинаковых персонажей.
  3. Выберите трассу и нажмите клавишу ENTER.

TIME TRIAL (ТРЕНИРОВКА)

В этом разделе вы можете ознакомиться со всеми трассами и потренироваться, прежде чем начать Большое приключение. Войдя в этот раздел, вы увидите меню начала игры.

Внимание! В разделе TIME TRIAL на экране выбора игрового уровня вы сразу можете выбрать любую трассу.

МЕНЮ НАЧАЛА ИГРЫ

  • NEW GAME (Новая игра)
    Чтобы начать новую игру, выберите NEW GAME.
    Перед началом заезда вам предстоит выбрать гонщика. Для просмотра всех персонажей пользуйтесь клавишами СТРЕЛКАМИ. Чтобы выбрать персонажа нажмите ENTER. Затем введите свое имя. Чтобы выбрать букву из списка, пользуйтесь клавишами СТРЕЛКАМИ. Для выбора буквы нажмите ENTER. Для отмены введенной буквы используйте кнопку Del (Удалить) в нижней части экрана. Когда закончите, нажмите внизу экрана Ok. Чтобы вернуться на предыдущий экран, нажмите клавишу BACKSPACE.
    Выбрав гонщика, вы увидите экран, на котором собрана вся информация об этом участнике. По мере того как вы будете находить детали и завоевывать призы, пустые места в окне будут заполняться. Чтобы перейти на экран выбора игрового уровня, нажмите Ok внизу экрана. Чтобы вернуться на предыдущий экран, нажмите клавишу BACKSPACE.
    Для выбора уровня пользуйтесь клавишами СТРЕЛКАМИ.
  • LOAD GAME (Загрузить игру)
    Здесь вы можете восстановить сохраненную игру. Для этого выберите LOAD GAME, а в появившемся списке – нужную игру.
  • GO BACK (Вернуться)
    Чтобы вернуться в главное меню, выберите GO BACK.

BEST TIMES (ЛУЧШИЕ РЕЗУЛЬТАТЫ)

В этом разделе вы можете ознакомиться со статистикой игроков, сохранивших свои игры, а также с лучшими результатами. Чтобы просмотреть статистику игрока, выберите строку с его именем. Чтобы узнать лучшие результаты, выберите строку BEST OF BEST. Чтобы вернуться в главное меню, выберите GO BACK.

OPTIONS (НАСТРОЙКИ)

  • SOUND (Звуковое сопровождение), VOICE (Голос), MUSIC (Музыка)
    Вы можете настроить громкость этих параметров. Звук регулируется в диапазоне от 1 до 11 единиц.
  • DISPLAY (Экран)
    Здесь вы можете выбрать драйвер монитора (DISPLAY DEVICE), а также разрешение экрана и глубину цвета (SCREEN RESOLUTION). Чтобы вернуться на предыдущий экран, выберите DONE.
  • CONTROLS (Клавиши управления)
    В этом разделе вы можете настроить клавишу управления. Для управления используется либо клавиатура, либо игровой пульт. Чтобы вернуться на предыдущий экран, выберите DONE.
  • VIBRATION ON/OFF (Вибрация Вкл/Выкл)
    Данная настройка появляется только при подключении игрового пульта.
  • DONE (Готово)
    Чтобы вернуться в главное меню, выберите строку DONE.

QUIT GAME

Выберите этот пункт, если хотите выйти из игры.

СОХРАНЕНИЕ И ЗАГРУЗКА ИГРЫ

Сохранение

После окончания заезда вы перейдете на экран с результатами гонки. Затем появится меню, где вы можете выбрать один из трех пунктов:

  • Continue (Продолжить) – вернуться в игру без сохранения.
  • Save Game (Сохранить игру) – сохранить игру.
  • Options (Настройки) – изменить настройки игры.

Загрузка

Восстановить сохраненную игру можно из меню начала игры.
